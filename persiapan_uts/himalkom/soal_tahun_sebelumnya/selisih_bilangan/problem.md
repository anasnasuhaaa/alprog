# Menghitung Jumlah Pasangan dengan Selisih Tertentu

Diketahui terdapat **n bilangan bulat tidak negatif** yang tersusun **tidak menurun**, serta sebuah bilangan **S ≥ 0**.  
Tugas Anda adalah membuat algoritme (pseudocode) atau program C untuk menentukan **banyaknya pasangan bilangan (a, b)** yang memenuhi:
b - a = S

## Input Format
Baris pertama berisi dua bilangan bulat: n dan S  
Baris kedua berisi n bilangan bulat tidak negatif yang tersusun tidak menurun, dipisahkan oleh spasi

## Constraints
1 ≤ n ≤ 10000  
0 ≤ S

## Output Format
Sebuah bilangan bulat yang menyatakan jumlah pasangan (a, b) dengan selisih S

## Sample Input
5 2  
5 7 7 10 12

## Sample Output
3

## Penjelasan
Pasangan yang memenuhi:
(5, 7)  
(5, 7)  
(10, 12)

Sehingga total pasangan adalah 3.
